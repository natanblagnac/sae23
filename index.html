SAE23 - Gestion des données de capteurs

Projet universitaire pour la gestion des mesures environnementales

Bienvenue sur la plateforme SAE23

Ce site permet de consulter et gérer les données de capteurs (température, humidité, CO2) collectées dans deux bâtiments. Optimisé pour le Fairphone 3+, il respecte les principes d'écoconception pour une faible consommation énergétique.
